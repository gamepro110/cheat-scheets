# pacman

## sync and update all packages

```bash
sudo pacman -Syu
```

| flag       | description                                                 |
| ---------- | ----------------------------------------------------------- |
| `-S`       | to be determined                                            |
| `-y`       | sync with mirrors                                           |
| `-u`       | upgrade packages                                            |
| `--ignore` | list of packages to ignore on this run, separated by commas |
